## Quick Start

### Step 1: Clone the Repository

```bash
git clone https://github.com/rikitrader/federal-trial-counsel.git
cd federal-trial-counsel
```

### Step 2: Run the Unlock Script

```bash
./scripts/unlock.sh
```

### Step 3: Enter Your Password

When prompted, enter the password provided by the repository owner.

### Step 4: Skill Auto-Installs

The skill will be automatically installed to `~/.claude/skills/federal-trial-counsel/` and will activate when you start Claude Code.

## Troubleshooting

**"Wrong password" error:**
- Verify you have the correct password from the repository owner
- Passwords are case-sensitive

**"Permission denied" error:**
```bash
chmod +x ./scripts/unlock.sh
```
