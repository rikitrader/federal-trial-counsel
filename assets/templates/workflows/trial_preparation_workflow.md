# FEDERAL TRIAL PREPARATION WORKFLOW

## Overview

Comprehensive checklist and workflow for preparing a federal civil case for trial in the Middle District of Florida.

---

## OBJECTION QUICK REFERENCE

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ COMMON TRIAL OBJECTIONS                                                     │
├─────────────────────────────┬───────────────────────────────────────────────┤
│ Objection                   │ When to Use                                   │
├─────────────────────────────┼───────────────────────────────────────────────┤
│ Relevance (401/402)         │ Evidence has no bearing on case              │
│ Hearsay (801/802)           │ Out-of-court statement for truth             │
│ Leading (611)               │ Suggesting answer on direct                  │
│ Speculation                 │ Witness guessing or assuming                 │
│ Lack of Foundation          │ No basis established for testimony           │
│ Calls for Narrative         │ Overly broad question                        │
│ Asked and Answered          │ Same question already posed                  │
│ Compound Question           │ Multiple questions in one                    │
│ Assumes Facts Not in Evidence│ Question contains unproven facts            │
│ Argumentative               │ Counsel arguing with witness                 │
│ Beyond Scope                │ Cross exceeds direct exam scope              │
│ Best Evidence (1002)        │ Original document required                   │
│ Prejudicial (403)           │ Prejudice outweighs probative value         │
│ Character Evidence (404)    │ Improper character evidence                  │
│ Expert Opinion (702)        │ Expert not qualified or reliable             │
└─────────────────────────────┴───────────────────────────────────────────────┘
```
